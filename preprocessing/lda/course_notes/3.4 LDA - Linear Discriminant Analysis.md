# 1. The Setting

We are given a **supervised learning dataset**:

- Inputs:  
  $x_i \in \mathbb{R}^d$, for $i = 1, \dots, N$
- Outputs (class labels):  
  $t_i \in \{1, \dots, k\}$

We collect:
- Inputs into a matrix  
  $X \in \mathbb{R}^{d \times N}$
- Labels into a vector  
  $t \in \mathbb{R}^N$

In classification, the ultimate goal is to estimate:
$$
P(C_j \mid x)
$$
for each class $j = 1, \dots, k$.

---
# 2. The Problem LDA Addresses

Instead of directly constructing a classifier, we first ask:
> Can we transform the data into a lower-dimensional space that makes classification easier?

Specifically:
- The original data lives in $\mathbb{R}^d$
- We want a projection into $\mathbb{R}^m$, where $m < d$
- The projection should preserve class separation

So we want a linear transformation:
$$
x \in \mathbb{R}^d \longrightarrow z \in \mathbb{R}^m
$$

such that the classes remain well separated in the reduced space.

---
# 3. Why Not Just Use PCA?

## PCA Objective

PCA finds directions that maximize **total variance** in the data:

$$
\text{Maximize total variance of projected data}
$$

However:
- High variance does not necessarily imply good class separation.
- A feature can vary significantly but still be irrelevant for distinguishing classes.

PCA is **unsupervised** — it ignores labels entirely.

---
# 4. LDA’s Core Idea

LDA is a **supervised dimensionality reduction technique**.

Instead of maximizing overall variance, LDA tries to:
> Find projection directions that maximize separation between classes.

More precisely, LDA aims to:
- Make class means far apart (maximize between-class variance)
- Make samples within each class tightly clustered (minimize within-class variance)

So the objective becomes:
$$
\text{Maximize between-class variance}
\quad \text{and} \quad
\text{Minimize within-class variance}
$$

---
# 5. Conceptual Difference: PCA vs LDA

| PCA | LDA |
|------|------|
| Unsupervised | Supervised |
| Ignores class labels | Uses class labels |
| Maximizes total variance | Maximizes class separability |
| Good for compression | Good for classification |

---
# 6. What This Section Is About

In this section, we are **not yet building a probabilistic classifier**.

Instead, we are:
- Learning a linear projection
- Reducing dimensionality
- Transforming the data
- Making classification easier in the projected space

This formulation is often called **Fisher’s Linear Discriminant**.
