# Stepwise Regression Selection

Simple forward/backward stepwise predictor selection for linear regression.

## Files

- `stepwise_regressor.py`: selector implementation
- `metrics.py`: scoring helpers (RSS, R2, adjusted-R2, AIC, BIC, Cp)
- `test_stepper.py`: quick synthetic-data script

## Quick run

- From repo root:
  - `python feature_selection/stepwise_regression_selection/test_stepper.py`
